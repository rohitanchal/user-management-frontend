This is a production-ready full-stack application built with Angular Standalone + RxJS on the frontend and Node.js + Express + MongoDB on the backend. It features secure JWT-based authentication, role-based access control (admin/user), and full CRUD operations for user management. Admins can log in, view all users, create new users with hashed passwords, update existing ones, and delete accounts. All API requests are protected using an HTTP Interceptor and restricted using middleware authorization. Designed for scalability, it supports millions of users using real-time reactive patterns with RxJS.
